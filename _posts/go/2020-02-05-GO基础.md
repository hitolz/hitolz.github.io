## 定义变量
使用var关键字,变量类型放在变量名称的后边。
```
定义一个type类型的变量
var variableName type
定义多个type类型的变量
var v1,v2,v3 type
定义变量并初始化
var variableName type = value
同时初始化多个变量
var v1,v2,v3 type = a1,a2,a3
```
可以忽略类型
```
var v1,v2,v3 = a1,a2,a3
```

简短声明，可以省略var关键字。只在函数内部使用，定义全局变量还得使用var
```
v1,v2,v3 := a1,a2,a3
```

## 基础类型
### bool
值是true或false
### 数值类型
#### 整数

| 类型 | 长度 |
| --- | --- |
| int8 | 1byte |
| int16 | 2byte |
| int32 | 4byte |
| int64 | 8byte |
| int | 根据系统位数，4byte或者8byte |
| uint8 | 1byte |
| uint16 | 2byte |
| uint32 | 4byte |
| uint64 | 8byte |
| uint | 根据系统位数,4byte或8byte |
| byte | 1byte |
| rune | 4byte |

假设在某台机器上，int的长度是4byte，但int与int32不能互用。

#### 浮点数
类型有float32和float64两种，默认float64
#### 复数
complex64（32位实数+32位虚数）和complex128（64位实数+64位虚数）

### string
字符串基于UTF-8编码
1. 字符串是一个不可改变的字节序列，通过双引号或反引号定义，它的类型是string
2. 可以包含任何数据，包括byte和0
3. 是字节的定长数组
4. 底层就是Byte数组
5. 字符串定义之后不可以直接修改
